## ğŸ“š Documentation

### Prerequisites

- Node.js and npm installed.
- Git installed.

### Setup

1. **Clone the repository:**

   ```sh
   git clone https://github.com/itsharta/itsharta.github.io.git
   ```

2. **Install dependencies:**

   ```sh
   cd itsharta.github.io
   npm install
   ```

3. **Run the development server:**

   ```sh
   npm run dev
   ```

### Usage

- **Build for production:**

  ```sh
  npm run build
  ```

- **Deploy:**

  ```sh
  npm run deploy
  ```
